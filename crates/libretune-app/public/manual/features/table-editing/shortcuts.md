# Keyboard Shortcuts

Complete reference for table editing keyboard shortcuts.

## Navigation

| Shortcut | Action |
|----------|--------|
| Arrow Keys | Move selection |
| Tab | Move to next cell |
| Shift+Tab | Move to previous cell |
| Home | Go to first cell in row |
| End | Go to last cell in row |
| Ctrl+Home | Go to first cell in table |
| Ctrl+End | Go to last cell in table |
| Page Up | Scroll up one page |
| Page Down | Scroll down one page |

## Selection

| Shortcut | Action |
|----------|--------|
| Shift+Arrow | Extend selection |
| Ctrl+A | Select all cells |
| Ctrl+Click | Toggle cell in selection |
| Shift+Click | Extend selection to cell |
| Escape | Clear selection |

## Editing

| Shortcut | Action |
|----------|--------|
| Enter | Edit selected cell |
| Delete | Clear selected cells |
| Backspace | Clear and edit |
| + or > | Increase value by step |
| - or < | Decrease value by step |
| Shift + (+/-) | Increase/decrease by 10x step |
| = | Set equal (average) |
| * | Scale by factor |
| / | Interpolate |
| S | Smooth selection |

## Clipboard

| Shortcut | Action |
|----------|--------|
| Ctrl+C | Copy selection |
| Ctrl+V | Paste |
| Ctrl+Shift+V | Paste special |
| Ctrl+X | Cut selection |

## History

| Shortcut | Action |
|----------|--------|
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+Shift+Z | Redo (alternate) |

## View

| Shortcut | Action |
|----------|--------|
| Ctrl+0 | Zoom to fit |
| Ctrl++ | Zoom in |
| Ctrl+- | Zoom out |
| F | Toggle follow mode |
| 3 | Toggle 3D view |

## File

| Shortcut | Action |
|----------|--------|
| Ctrl+S | Save tune |
| Ctrl+B | Burn to ECU |
