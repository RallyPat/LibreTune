# Global Keyboard Shortcuts

Complete reference for LibreTune keyboard shortcuts.

## File Operations

| Shortcut | Action |
|----------|--------|
| Ctrl+N | New project |
| Ctrl+O | Open project / Load tune |
| Ctrl+S | Save tune |
| Ctrl+Shift+S | Save tune as |
| Ctrl+B | Burn to ECU |
| Ctrl+Q | Quit application |

## ECU Connection

| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+C | Connect to ECU |
| Ctrl+Shift+D | Disconnect from ECU |

## Navigation

| Shortcut | Action |
|----------|--------|
| Ctrl+1-9 | Switch to tab 1-9 |
| Ctrl+Tab | Next tab |
| Ctrl+Shift+Tab | Previous tab |
| Ctrl+W | Close current tab |
| F11 | Toggle full screen |

## Table Editor

| Shortcut | Action |
|----------|--------|
| Arrow keys | Navigate cells |
| Enter | Edit cell |
| Delete | Clear cell |
| = | Set equal (average) |
| * | Scale selection |
| / | Interpolate |
| S | Smooth selection |
| Ctrl+A | Select all |
| Ctrl+C | Copy |
| Ctrl+V | Paste |
| Ctrl+Shift+V | Paste special |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| F | Toggle follow mode |
| 3 | Toggle 3D view |

## AutoTune

| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+A | Open AutoTune |
| Space | Start/Stop AutoTune |

## Dashboard

| Shortcut | Action |
|----------|--------|
| Ctrl+D | Switch to dashboard |
| Escape | Exit designer mode |

## Data Logging

| Shortcut | Action |
|----------|--------|
| Ctrl+L | Start/Stop logging |

## Help

| Shortcut | Action |
|----------|--------|
| F1 | Open help |
| Ctrl+? | Keyboard shortcuts |

## Dialogs

| Shortcut | Action |
|----------|--------|
| Escape | Close dialog |
| Enter | Confirm dialog |
| Tab | Next field |
| Shift+Tab | Previous field |
